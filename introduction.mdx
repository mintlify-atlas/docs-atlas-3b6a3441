## What is SESA?

**SESA (Sistema de Evaluación y Seguimiento Académico)** is a full-stack academic management platform designed to centralize school management for Universidad Interamericana para el Desarrollo (UNID), Campus Campeche. The platform automates student tracking, digitalizes evaluation processes, and provides real-time academic progress information.

Unlike manual methods, SESA eliminates information fragmentation and human errors through a robust and scalable infrastructure.

## Technology Stack

SESA is built with modern, production-ready technologies:

### Frontend
- **React** - Component-based user interfaces
- **Vite** - Lightning-fast development and build tool
- **Tailwind CSS** - Utility-first CSS framework for responsive design
- **Axios** - HTTP client for API communication

### Backend
- **Python** - Data processing and business logic
- **FastAPI** - High-performance API framework with automatic OpenAPI documentation
- **SQLAlchemy** - SQL toolkit and ORM
- **APScheduler** - Background task scheduling for data synchronization

### Database & Infrastructure
- **MySQL** - Relational database for data integrity
- **Docker** - Containerized database deployment
- **Tailscale** - Secure mesh VPN for team collaboration
- **Netlify** - Continuous deployment for the frontend
